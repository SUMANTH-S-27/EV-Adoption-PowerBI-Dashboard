# Electric Vehicle Adoption Analysis Dashboard

This project analyzes electric vehicle adoption trends in India from 2015 to 2025 using Power BI.

## Tools Used
- Power BI
- CSV Dataset
- Data Visualization

## Dashboard Insights
- EV adoption has increased rapidly after 2020.
- Two-wheelers dominate the EV market in India.
- EV market penetration is growing steadily each year.

## Files
- EV_Adoption_Dashboard.pbix – Power BI dashboard
- Indian_EV_Sales_2015_2025.csv – Dataset
- dashboard.png – Dashboard preview
- EV_Project_Presentation.pptx – Project presentation
